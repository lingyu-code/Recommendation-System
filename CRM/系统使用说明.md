# 金融智能推荐系统使用说明

## 系统概述

本系统是一个基于Django后端和Vue前端的金融智能推荐系统，提供保险、基金、股票的个性化推荐服务。

## 系统架构

- **后端**: Django + Django REST Framework + SQLite
- **前端**: Vue 3 + Element Plus + ECharts
- **推荐算法**: KNN、余弦相似度、协同过滤、行业相关性分析、趋势分析

## 快速启动

### 方法一：使用启动脚本（推荐）
双击运行 `start_system.bat` 文件，系统将自动启动后端和前端服务。

### 方法二：手动启动

1. **启动后端服务**：
   ```bash
   cd backend
   python manage.py runserver
   ```
   后端服务将在 http://127.0.0.1:8000 运行

2. **启动前端服务**：
   ```bash
   cd frontend
   npm run dev
   ```
   前端服务将在 http://localhost:5173 运行（端口可能不同）

## 系统功能

### 1. 投资中心
- **仪表盘**: 显示投资概览和推荐产品
- **基金推荐**: 基于用户画像的个性化基金推荐
- **股票推荐**: 基于行业分析和趋势预测的股票推荐

### 2. 理财规划
- **理财诊断**: 分析用户财务状况并提供建议
- **保险推荐**: 基于用户需求的保险产品推荐

### 3. 个人中心
- **用户信息管理**
- **投资组合查看**
- **浏览历史记录**

## API接口

访问 http://127.0.0.1:8000/ 查看所有可用API端点：

### 主要API端点：
- `GET /api/user-profiles/` - 用户信息
- `GET /api/insurances/` - 保险产品列表
- `GET /api/funds/` - 基金产品列表  
- `GET /api/stocks/` - 股票列表
- `GET /api/insurance/recommended/?user_id=1` - 保险推荐
- `GET /api/fund/recommended/?user_id=1` - 基金推荐
- `GET /api/stock/recommended/?user_id=1` - 股票推荐
- `POST /api/financial-diagnosis/` - 理财诊断

## 推荐算法

系统采用多种推荐算法：
1. **KNN算法**: 基于用户相似度的推荐
2. **余弦相似度**: 产品特征相似度计算
3. **协同过滤**: 基于用户行为的推荐
4. **行业相关性分析**: 股票行业关联度分析
5. **趋势分析**: 基于历史数据的趋势预测


### 数据库管理
```bash
cd backend
python manage.py migrate  # 应用数据库迁移
python manage.py createsuperuser  # 创建管理员账户
```

## 技术特点

1. **模块化设计**: 前后端分离，API驱动
2. **多样化推荐**: 多种算法组合，提高推荐准确性
3. **实时交互**: 基于用户行为的动态推荐
4. **数据可视化**: 使用ECharts展示投资数据
5. **响应式界面**: 适配不同设备屏幕

## 故障排除

### 常见问题

1. **后端404错误**: 检查Django服务器是否正常启动
2. **前端依赖问题**: 运行 `npm install` 安装依赖
3. **数据库问题**: 运行 `python manage.py migrate` 应用迁移
4. **端口冲突**: 修改settings.py中的端口配置

### 日志查看
- 后端日志: Django服务器控制台输出
- 前端日志: Vue开发服务器控制台输出
